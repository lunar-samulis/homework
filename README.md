# Prerequisites
npm

# Running app

Copy given Github credentials (or use yours) to field `AppSettings.GITHUB_CREDENTIALS` 
as `username:password`, then:

`npm install`

`npm start`

# Usage

`npm start` should open <http://localhost:4200/list>.
